# 📚 Study Tracker App

A full-stack productivity web application built to help students track their study time, manage daily goals, take notes, and stay consistent using a simple and interactive dashboard.

---

## 🚀 Features

- 🔐 User Login System (Spring Boot backend)
- ⏱ Pomodoro Timer (Start / Stop / Reset)
- 🎯 Daily Study Goals Tracker
- 📝 Notes Section
- 👤 User Profile Management
- 📊 Study Progress Dashboard
- 🔥 Streak Tracking System
- 📈 Weekly Study Graph Visualization

---

## 🛠 Tech Stack

### Frontend:
- HTML5
- CSS3
- JavaScript (Vanilla)

### Backend:
- Java
- Spring Boot
- REST API

### Database (Future Scope):
- MongoDB / MySQL
